# 📚 Library Management System

A Python-based Library Management System with a menu-driven interface, designed using dictionary-based data structures and modular programming. This system allows efficient handling of book records, issuing, returning, and fine calculation.

---

## 🚀 Features

- ➕ Add new books
- 📖 View all available books
- 📤 Issue books to students
- 📥 Return books
- 💰 Fine calculation for late returns
- 🧠 Dictionary-based data storage
- 🔁 Menu-driven system using loops
